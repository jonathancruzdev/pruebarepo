# Pasos para crear un repositorio
1. Inicializo un repositorio
    ``` bash
        git init
    ```
2. Configuro el usuario & email
    ``` bash
        git config user.name "John Do"
        git config user.mail "johndo@mail.com"

    ```
3. Agrego los archivos
    ``` bash
        git add main.js
        git add .
    ```


4. Creo el commit 
5. Creo el repositorio
6. Realizo el push
